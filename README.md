# 🐳 Docker Bootcamp by TeraSkill

🚀 A structured, day-wise Docker learning repository designed to take you from beginner to advanced with hands-on practice.

---

## 📌 What You'll Learn

* Docker fundamentals
* Containers vs Virtual Machines
* Docker CLI & commands
* Writing Dockerfiles
* Building & managing images
* Docker registry
* Docker volumes & networking
* Multi-container applications
* Docker Swarm
* Docker Scout
* Real-world DevOps use cases

---

## 📂 Repository Structure

Each day contains:

* 📖 `notes.md` → Concept explanations
* 🛠️ `tasks.md` → Hands-on exercises

Example structure:

```
docker-bootcamp/
│
├── Day-01/
│   ├── notes.md
│   ├── tasks.md
│
├── Day-02/
│   ├── notes.md
│   ├── tasks.md
│
├── Day-03/
│   ├── notes.md
│   ├── tasks.md
│
└── README.md
```
